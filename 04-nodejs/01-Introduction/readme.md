### Is web brwoser JavaScript runtime?

- The JavaScript Runtime (V8) can do one thing at a time.
- But the _Browser_ is more than a runtime. It gives us:
  - WebAPIs [DOM - ajax - setTimeout]
  - Event Loop [callback queue]
